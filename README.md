# Login & Registration System

## Overview

This project is a Login and Registration System developed using Python, Flask, MySQL, HTML, CSS, and JavaScript. It allows users to register, log in, and log out securely.

## Features

- User Registration
- User Login
- User Logout
- Password Hashing
- Session Management
- MySQL Database Integration
- Responsive User Interface

## Technologies Used

- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap
